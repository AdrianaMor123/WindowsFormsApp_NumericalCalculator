# 🧮 Calculator App

A simple desktop calculator built with **C#** and **Windows Forms** in Visual Studio.

---

## ✨ Features

- Basic arithmetic operations — addition, subtraction, multiplication, division
- Correct **order of operations** (multiplication and division evaluated before addition and subtraction)
- Decimal number support
- Negate a number (toggle positive/negative)
- Backspace to delete the last digit
- Clear to reset
- Custom **draggable title bar** (borderless window you can drag around)

---

## 🛠️ Built With

- **C#** — .NET Windows Forms
- **Visual Studio**

---

## 🚀 Getting Started

### Prerequisites
- Windows OS
- [Visual Studio](https://visualstudio.microsoft.com/) with the **.NET Desktop Development** workload installed

### Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/YOUR_USERNAME/Calculator.git
   ```
2. Open `Calculator.sln` in Visual Studio
3. Press **F5** or click **Start** to build and run
